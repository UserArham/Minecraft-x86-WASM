# Minecraft Web (Eaglercraft Wrapper)

![License: MPL 2.0](https://img.shields.io/badge/License-MPL%202.0-orange.svg)

A fullscreen web wrapper for **Eaglercraft 1.12.2 (WASM)** with a responsive loading screen that scales to fit any screen size.

This project embeds Eaglercraft in an iframe and adds UI features such as a fullscreen loading screen. No Minecraft assets are included or redistributed.
